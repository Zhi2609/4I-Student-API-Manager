# 4I-Student-API-Manager
---
Este proyecto es una aplicación web didáctica que demuestra la integración de la Programación Orientada a Objetos (POO) en JavaScript para la gestión de datos escolares. El sistema implementa una arquitectura desacoplada donde el frontend (HTML/CSS) interactúa con una lógica de negocio que emula el comportamiento de una API RESTful.

El programa permite administrar un grupo de alumnos, realizar consultas generales y ejecutar eliminaciones lógicas, todo manteniendo un estado en memoria durante la sesión del navegador.
